Features

The [CODE + PICKLE] Platform

The CP platform is the quickest way for a company to develop and deploy apps that immediately start producing value.

The powerful CP Dashboard rich ecosystem of managed data services includes a secure and easy to navigate environment. We believe developers should have immediate access to a scalable, highly available database with rollback - one that supports their apps and development style.

Using an existing, high-quality library of components is something that empowers developers - they can build more, faster, by using trusted code that provide the functionality that they require. All of our components and add-ons are fully-managed services, integrated for use with Code+Pickle. They can be provisioned and scaled in one command, and let developers extend the capabilities of an application. Over 200 Add-ons and over 1,000 snippets are available, providing services ranging from databases to uptime alerts, messaging services or automatic backups to search, metrics or mail delivery.

The CP platform allows you the flexibility to be able to deploy from Git, GitHub, or Docker, or using an API. Plug into the most popular CI systems and servers for consistent and automated application delivery.

You have complete control of all your apps in one place. This is accomplished with the use of smart containers, which are part of an elastic runtime platform that provides orchestration, load balancing, logging and security.

The powerful CP+ Shield gives you enhanced trust controls, such as Private Space Logging, that greatly simplify compliance auditing while still giving you full control of app configuration and deployment.

Code+Pickle unifies the data in your Postgres database with the contacts, accounts and other custom objects in the database. Easily configured with a point and click UI, it’s simple to get the service up and running in minutes – no coding or complex configuration is required.
